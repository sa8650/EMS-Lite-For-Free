User Manual: Enabling All ActiveX Controls for using EMS 
⚠️ Warning:
This setting is not recommended for general users as it may allow potentially unsafe controls to run. Use this only in trusted environments.
✅ Steps to Enable All ActiveX Controls:
1.	1. Open Excel.
2.	2. Go to the top menu bar and click:
   File ➝ Options
3.	3. In the Excel Options window, click:
   Trust Center ➝ Trust Center Settings...
4.	4. In the Trust Center window, from the left menu, select:
   ActiveX Settings
5.	5. On the right side, select the option:
   Enable all controls without restrictions and without prompting (not recommended; potentially dangerous controls can run)
6.	6. (Optional): Make sure the checkbox for:
   Safe mode (helps limit the control's access to your computer) is unchecked for full unrestricted access.
7.	7. Click OK to close the Trust Center window.
8.	8. Click OK again to close the Excel Options window.

📌 Your settings are now updated!
now u can running EMS without any ActiveX error.
